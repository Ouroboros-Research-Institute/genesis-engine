Genesis Engine is a minimal computational model that asks whether biological agency must be assembled in a particular order. Clock → Map → Agency · ordered in 482 of 482 runs · N = 500 seeds
THE CLAIM

We define three sequential observables on an evolving protocell population: a Clock (regular division), a Map (spatially stable pattern), and Agency (the two persisting jointly across generations). In 482 of 482 independent simulations where both Clock and Map were reached, Clock latched first. The binomial p-value is 8.01 × 10⁻¹⁴⁶; the Wilcoxon signed-rank p is 1.66 × 10⁻⁹⁸. Hedges' g between stable and disorganized populations is 9.41.

That is the only claim. Everything below is the bookkeeping that makes it testable.

THE FOUR PHASES
PHASEPREDICATEMEANING
A PREBIOTIC default No transition has latched yet.
B CLOCK CV < 0.25 Division intervals become regular.
C MAP (after B)   S̄ > 0.25 Stable spatial pattern held across the population.
D AGENCY (after C)   S̄ > 0.35 ∧ CV < 0.30 ∧ gen ≥ 5 Clock and Map persist across at least five lineal generations.

Thresholds are observer-imposed; the ordering is robust to ±30% perturbation of every cutoff — see the ablation table on the Monte Carlo tab.

PATTERN STABILITY · S

S is the population-mean coherence of the reaction-diffusion v-field across divisions. Each daughter inherits a perturbed copy of the parent's field; if the pattern survives that perturbation and the subsequent RD relaxation, S → 1. If the pattern randomizes, S → 0. Computed as the maximum normalized circular cross-correlation between parent and daughter v-vectors, averaged over the living population.

S captures heritable spatial structure, which is what makes a pattern a "map" rather than an accident.

DIVISION CV

CV = σ/μ of the last three division intervals, averaged across cells that have completed ≥3 divisions. A cell that divides on a steady beat has CV → 0; a cell that divides irregularly has CV → 1.

CV is capped at 1.0 during startup (fewer than three divisions yet) — that is why CV=1.000 is so common in the first ~2000 ticks of a fresh 2D run.

LATCHED TRANSITIONS vs CURRENT STATE
LATCHED

Each phase latches at the first tick its predicate is satisfied, given the ordering constraint (B before C, C before D, D additionally requires gen ≥ 5). Once latched, always latched — even if the population briefly stops satisfying the predicate later. This is the quantity the Monte Carlo tabulates.

CURRENT STATE

Shows whether the predicate is satisfied right now, with explicit gate markers (CV < 0.25 ✓, CV < 0.30 ✗, …). A population can hit Map, then drift back under the threshold: LATCHED still shows C, CURRENT STATE shows B. That drift is real biology, not a bug.

The paper's statistic asks "did Clock come before Map?", not "is Clock currently holding?". Splitting the two displays is what keeps that distinction honest.

READING THE THREE TABS
LIVE 1D

24-node ring membrane, RD_STEPS = 3, MAX_CELLS = 100. Fast, deterministic-looking, usually reaches phase D within ~6000 ticks. This is the configuration the N=500 Monte Carlo uses.

LIVE 2D SPHERE

642-vertex icosphere (sub = 3), cotangent Laplacian, RD_STEPS = 90, α = 0.40, MAX_CELLS = 30. Spatially expressive but sensitive to seed — some runs reach D by ~5500 ticks, others get trapped with high S but CV plateau. Use CURRENT STATE to see why a given seed stalls.

MONTE CARLO

Pre-computed batched 1D runs (N = 500, 80 000 ticks each) aggregated offline. This tab carries the statistical weight — the live tabs are existence proofs, not evidence.

WHAT THIS MODEL IS NOT
  • Not a faithful protocell. The membrane is a ring or sphere with two chemicals (u, v) and a division-on-growth rule. No metabolism, no real genome, no chemistry.
  • Not a claim about chemistry. The ordering result is compositional — how temporal regularity and spatial stability assemble — not a prediction about specific abiogenesis pathways.
  • Live 2D ≠ statistical evidence. The MAX_CELLS = 30 cap is for real-time rendering. Single seeds can stall. Ordering is proven across the Monte Carlo ensemble, not within any one live run.
  • Observer-imposed thresholds. B_CV = 0.25, C_S = 0.25, D_S = 0.35, D_CV = 0.30, D_GEN = 5. Robust to ±30% perturbation; see ablation table.
